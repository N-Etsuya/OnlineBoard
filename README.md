## オンライン掲示板システム

### 概要
- クライアントサーバー型のオンライン掲示板システム
- ライブラリ、言語: Java FX、Java、SQLite
- 機能: ユーザ登録、ログイン、スレッド作成、スレッド一覧取得、スレッド削除、ポスト作成、ポスト一覧取得、ポスト削除
- 大学の講義の一環として、ハッカソン形式で開発

### 動作環境

- Java実行環境 (JRE) 17 以上

### 実行方法

1. **バックエンドの起動:**

   以下のコマンドを実行してバックエンドサーバーを起動する。

   ```bash
   java -jar backend/target/backend-1.0-SNAPSHOT-jar-with-dependencies.jar
   ```

2. **フロントエンドの起動:**

   以下のコマンドを実行してフロントエンドアプリケーションを起動する。

   ```bash
   frontend/bin/online-board
   ```
